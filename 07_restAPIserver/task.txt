Design REST API -JSON based 

Task to perform using REST API, cross platform server for browser send data in html and for other device send json

GET /users - List all users in HTML document
GET /api/users - List all users in JSON format
GET /api/users/n - Get the User with ID n passed

Dynamic Path Parameter
GET /api/users/:id  --> where id is the dynamic variable

POST /api/users - Create new users

PATCH /api/users/Id_number - Edit the user with Id_number passed

DELETE /api/users/Id_number - Delete the user with Id_number passed

Hardcoding the user data from the site mockaroo.com generates fake data for testing purposes
